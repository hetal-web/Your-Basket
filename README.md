# YourBasket - Grocery Web App

## Objective
Build a simple frontend-based grocery system where users can manage items in a shopping cart.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Add items to cart
- Remove items from cart
- Dynamic cart updates
- Feedback form for user input
- Simple and user-friendly interface

## Description
YourBasket is a basic web application that simulates a grocery shopping experience. Users can add or remove items from their cart and also provide feedback through a form, demonstrating interactive frontend functionality.

## Outcome
Improved understanding of DOM manipulation, event handling, and user interaction handling through forms.
